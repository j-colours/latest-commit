# Latest Commit

Command line tool that simply grabs the most recent commit from the log (includes information and the 1-2 line commit message).

# Instructions

To run simply install it first where it will copy the tool onto your systems path. Keep in mind this works currently for Unix Systems only so on Windows
run it using `wsl`. ***MUST HAVE SUDO PERMISSIONS***. But can install using the `install` script to install it for you and `remove` to remove it when changing a version. Once installed just go into the git repo (directory) and run `lcommit` and it will run!
